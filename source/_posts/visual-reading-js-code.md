# 3. 怎么做呢?
## 3.1. step1:
```
npm install -g lambda-view
```
## 3.2. step2:
```
// 阅读本地代码
lv targer.js

// 阅读远程代码
lv https://code.jquery.com/jquery-3.1.1.js

// 阅读多个代码
lv file1.js file2.js file3.js

// 阅读所有
lv *.js
```
